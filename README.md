# Inheritance in C++

This repository contains a presentation, documentation, and C++ source code examples explaining the concept of inheritance in Object-Oriented Programming.

## Types of Inheritance Covered
- Single Inheritance
- Multiple Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance
- Method Overriding (Polymorphism)

## Folder Structure
Inheritance-Presentation/
├── slides/
├── documents/
├── examples/
└── images/


## How to Run the C++ Examples
1. Open any `.cpp` file in the `examples` folder.
2. Compile using:
g++ filename.cpp -o output

3. Run it:

./output
